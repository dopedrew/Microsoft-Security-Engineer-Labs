# Naming Conventions (Lab)

## Sentinel
- Analytic rule: `AR - <Technique> - <Scenario> - v<Major.Minor>`
- Hunting query: `HQ - <Technique> - <Scenario> - <Author>`

## Playbooks
- `PB - <Action> - <Trigger> - v<Major.Minor>`

## Files
- KQL: `TTP-<technique>-<scenario>.kql`
- Case studies: `cs-##-title.md`
