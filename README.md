This repo documents my hands-on security engineering labs focused on:
- Microsoft Sentinel (SIEM/SOAR)

- Microsoft Entra ID (Identity & Access)
- Automation (PowerShell, KQL, Logic Apps)

- **Automation:** Sentinel playbooks (Logic Apps) + response scripts

- **Case Studies:** End-to-end incident walkthroughs with lessons learned
